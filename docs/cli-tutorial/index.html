Command-Line Interface Tutorial

This is a tutorial for the command-line interface (CLI) for CiderPress II. With it, you can do roughly the same set of things that you can with the graphical user interface (GUI). It has the advantage that you can execute commands from shell scripts or other applications.

The interface offers more than 30 commands. This tutorial will demonstrate the use of the most common. For the rest, please refer to the full manual.
