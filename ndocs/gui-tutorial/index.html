Desktop GUI Tutorial

This is a tutorial for the graphical user interface (GUI) portion of CiderPress II, specifically the version developed for Windows. It has roughly the same set of features as the command-line interface, but for certain operations it can be significantly more convenient to use.

The tutorial covers common operations. For a full explanation of all features, please refer to the full manual.

All of the sample files used in this tutorial are available from the github source code repository, in the TestData directory, should you wish to try them for yourself.

Disk images and file archives are different things. Sometimes this tutorial refers to them collectively as "archives" for the sake of brevity.
