CiderPress II is a software tool for working with vintage Apple software, with an emphasis on the Apple II series of computers. It is the successor to the CiderPress utility.

Key features:

  • Support for a variety of vintage archive formats, as well as file attribute and resource fork preservation in modern ZIP archives.
  • Support for many disk image formats and filesystems, including multi-partition disk layouts.
  • Handles direct access to physical hard drives and removable media.
  • File converters for many document, graphic, and code files.

See this page for a complete list of supported file formats and conversions.

As part of this project, documentation was created for all of the supported archive, disk image, and vintage file formats. These can be accessed from the CiderPress Format Documentation Index.
